QuickClaw V3 — Start Here

Fast, external-drive-first OpenClaw setup for Mac. This page is for non-technical users and gives the exact click path.

Minimal clicks External SSD ready Local dashboard

First-time install (recommended path)

1
Open this QuickClaw folder in Finder.
2
Double-click QuickClaw_Setup.command.
3
When prompted for location, choose your external drive (for example: QUICKCLAWv2).
4
After install finishes, use scripts from the external drive path:
/Volumes/<DRIVE_NAME>/QuickClaw

Daily use

  • QuickClaw_Launch.command — starts gateway + dashboard
  • QuickClaw_Stop.command — stops QuickClaw services
  • QuickClaw_Verify.command — health checks
  • QuickClaw Doctor.command — diagnostics + logs

If macOS blocks script opening

Gatekeeper can block downloaded .command files. Use one of these:

  • Right-click the file → Open → confirm
  • or System Settings → Privacy & Security → Open Anyway

After first successful setup, this usually stops being an issue for this folder.

What success looks like

  • Launch output shows dashboard URL and gateway status
  • Dashboard opens and shows Gateway: RUNNING
  • Root path in dashboard points to your external drive QuickClaw folder

Fast connection setup (OpenAI + Telegram)

  • Go to Integrations tab in dashboard.
  • For OpenAI: paste API key, then use Quick Connect OpenAI.
  • For Telegram: get bot token from @BotFather, paste token, then use Telegram quick connect/save.
  • After connecting either one, use Apply Settings To Config.

Quick troubleshooting commands

bash /Volumes/QUICKCLAWv2/QuickClaw/QuickClaw_Stop.command

bash /Volumes/QUICKCLAWv2/QuickClaw/QuickClaw_Launch.command

tail -n 200 /Volumes/QUICKCLAWv2/QuickClaw/logs/dashboard.log

tail -n 200 /Users/<you>/.openclaw/logs/gateway.log
